[캠퍼스 네비게이션 실행 방법]

1. 압축 파일을 해제합니다.

2. 명령 프롬프트(cmd)를 실행합니다.

3. 아래 순서대로 입력합니다.

cd Desktop
cd campus_navi_final_updated_v4
cd campusfix
pip install flask << flask가 안깔려있을경우
python app.py

4. 실행 후 브라우저 주소창에 아래 주소를 입력합니다.

http://127.0.0.1:5000


[프로그램 주요 기능]

- 건물, 강의실, 편의시설 검색 가능
- 검색 결과에서 출발지 / 목적지 선택 가능
- 현재 위치 기반 안내 가능
- 지도에서 경로, 거리, 예상 시간 확인 가능
- 건물 클릭 시 내부 강의실 및 편의시설 정보 확인 가능


[폴더 구조 및 파일 설명]

campus_navi_final_updated_v4
└─ campusfix
   ├─ app.py
   │  : Flask 기반 메인 서버 파일
   │    CSV 데이터를 읽고 웹페이지 및 API를 실행하는 역할
   │
   ├─ classroom_data.csv
   │  : 강의실 정보 데이터 파일
   │    건물명, 호실코드, 호실명, 호실구분, 수용인원 등의 정보를 저장
   │
   ├─ building_coords.csv
   │  : 건물 좌표 데이터 파일
   │    각 건물의 위도(lat), 경도(lng) 정보를 저장
   │
   ├─ facilities.csv
   │  : 편의시설 정보 데이터 파일
   │    편의점, 카페, 식당 등 편의시설의 이름, 카테고리, 건물명, 층수 정보를 저장
   │
   ├─ README.TXT
   │  : 프로젝트 실행 방법과 파일 구조를 설명하는 문서
   │
   ├─ templates
   │  └─ index.html
   │     : 웹페이지의 전체 화면 구조를 담당하는 파일
   │       검색창, 지도 영역, 상세 정보 패널 등의 레이아웃을 구성
   │
   └─ static
      ├─ style.css
      │  : 웹페이지 디자인을 담당하는 스타일 파일
      │    버튼, 검색창, 지도 패널, 상세 정보 창 등의 화면 구성을 꾸밈
      │
      └─ script.js
         : 웹페이지 동작을 담당하는 자바스크립트 파일
           검색 기능, 지도 마커 표시, 현재 위치 반영, 경로 탐색,
           출발지 / 목적지 선택 등의 기능을 처리


[동작 방식 요약]

1. app.py가 CSV 파일들을 읽어 건물, 강의실, 편의시설 데이터를 불러옵니다.
2. index.html이 기본 웹 화면을 구성합니다.
3. script.js가 검색, 지도 표시, 경로 탐색 등의 동작을 처리합니다.
4. style.css가 전체 UI를 꾸며줍니다.
5. 브라우저에서 사용자가 검색하거나 현재 위치 버튼을 누르면,
   해당 정보가 지도 위에 반영됩니다.


[주의사항]

- 반드시 압축을 해제한 뒤 실행해야 합니다.
- app.py가 있는 campusfix 폴더에서 실행해야 정상 동작합니다.
- 현재 위치 기능은 브라우저에서 위치 권한 허용이 필요합니다.
- 실행이 되지 않을 경우, Python이 설치되어 있는지 먼저 확인해야 합니다.
